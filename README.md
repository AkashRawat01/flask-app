Flask app and docker-mongodb backend.

To run the frontend and backend follow the below steps:
1. Go to docker-mongodb folder and run  "docker compose up -d" This will download the docker mongodb image and run it in background.
2. Go to flask-tutorial folder > go to backend folder > run app.py using this cmd "python app.py". This will execute the backend.
3. Go to Frontend folder > run app.py using the same cmd "python app.py". This will execute the frontend.

   Test
   1. Go to your browser > go to "http://localhost:8000" fill the data or click on view data to access the database.
